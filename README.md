## Installation

### Clone the project
#### Via SSH
```
git clone git@github.com:adrien-nf/advent-of-code-2023.git
```
#### Via HTTPS
```
git clone https://github.com/adrien-nf/advent-of-code-2023.git
```

### Install modules
```
npm i
```

### Run a specific day
```
node 01/first-puzzle.js
```
